Desarrollo del proyecto

Primera Semana

La primera semana estuvimos pensando ideas para la realización del proyecto. Tuvimos problemas para entender el funcionamiento de GitHub. A partir de la segunda semana pudimos empezar a desarrollar la página web.

Segunda Semana

Comenzamos con el desarrollo de la página web con html y css todos juntos, en clase. Surgieron varios problemas.

Configuramos una función para mostrar y ocultar las diferentes ventanas. después, nos dimos cuenta de que era más sencillo tener varias páginas html e ir cambiando entre ellas mediante enlaces. Así que rehicimos esa parte durnate esa semana.

Tercera Semana

Continuamos desarrollando la página. Conseguimos darle forma a la página de los autores para que tuviese un diseño interactivo, al situarnos sobre las imágenes aparece el texto que describe a los autores.

Cuarta Semana

Empezamos a implementar el juego con canvas. Tenemos problemas para dibujar imágenes en el canvas pero acabamos entendiendo como se hace de manera correcta.

Tenemos dudas sobre como implementar de manera correcta el bucle del juego para conseguir la funcionalidad deseada.

Conseguimos el movimiento del personaje principal.

Quinta Semana

Seguimos desarrollando el juego. Reorganizamos algunas funciones y redefinimos los objetos para tener el código más limpio.

Conseguimos que haya colisiones del personaje principal con los objetos del juego. Ahora tendremos que afrontar el problema de tener varios objetos distintos a la vez en el tablero con un array.

Sexta Semana

Hemos tenido que cambiar la forma de declarar los objetos ya que necesitabamos que tuviesen metodos para poder desarrollar correctamente el juego y que el codigo fuese lo más sencillo posible. Al final los hemos declarado como clases con un constructor para que sea mas sencillo.

Además hemos creado 2 arrays con cada uno de los objetos que saldran y el jugador tendra que esquivar o coger dependiendo del objeto que sea. No sabemos como hacer para que cuando salgan tanto donuts como brocolis no salgan unos encima de otros.

Seguimos dando forma a la página web haciendo un menú desplegable en la parte de seguimiento y cambiando un poco los estilos de la página.

Creamos otra clase para introducir un objeto nuevo para tener más variedad en el juego.

Séptima Semana

Introducimos audio en el juego cuando colisionamos con los objetos.

Cambiamos las velocidades de los objetos para ir aumentando la dificultad del juego.

Cambiamos el fondo del juego mediante una clase nueva y un objeto nuevo.

Hacemos menús desplegables para los apartados de desarrollo y referencias.

Introducimos las fotos del apartado de autores.

Creamos una pantalla de finalización en la que nos indica la puntuación obtenida.
